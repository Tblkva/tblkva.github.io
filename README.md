# 🫀 Калькулятор для эхокардиографии

Современный веб-калькулятор для расчета параметров эхокардиографии с удобным интерфейсом и профессиональным выводом результатов.

## ✨ Особенности

- **🎯 Точные расчеты** - основан на медицинских формулах и стандартах
- **📱 Адаптивный дизайн** - работает на всех устройствах
- **🎨 Современный интерфейс** - темная тема с медицинской эстетикой
- **📋 Профессиональный вывод** - формат эхокардиографического заключения
- **📋 Копирование результатов** - удобное копирование в буфер обмена
- **⚡ Быстрая работа** - не требует установки, работает в браузере

## 🚀 Быстрый старт

1. Откройте `index.html` в любом современном браузере
2. Заполните обязательные параметры
3. При необходимости включите дополнительные секции
4. Нажмите "Рассчитать"
5. Скопируйте результаты при необходимости

## 📊 Параметры для расчета

### 🔴 Обязательные параметры

#### Размеры сердца
- **ЧСС** - Частота сердечных сокращений (уд/мин)
- **Вес** - Масса тела пациента (кг)
- **ЛП (диастола/систола)** - Левое предсердие (мм)
- **Аорта** - Размер аорты (мм)
- **ЛА** - Легочная артерия (мм)

#### Функция сердца
- **КДР/КСР** - Конечные диастолический и систолический размеры (мм)
- **КДО/КСО** - Конечные диастолический и систолический объемы (мл)
- **МЖП/ЗСЛЖ** - Межжелудочковая перегородка и задняя стенка ЛЖ (мм)

#### Венозный застой
- **КПВ** - Каудальная полая вена (мм)
- **ПВЛА** - Правая ветвь легочной артерии (мм)

### 🟡 Дополнительные параметры (опционально)

#### Клапанный аппарат
- **Митральный клапан** - E и A скорости (м/с)
- **Трикуспидальный клапан** - E и A скорости (м/с)
- **Аортальный клапан** - Скорость потока (м/с)
- **Клапан легочной артерии** - Скорость потока (м/с)

## 📈 Рассчитываемые показатели

### Основные соотношения
- **ЛП/Ао** - Соотношение левого предсердия к аорте
  ```
  ЛП/Ао = ЛПдиастола / Аорта
  ```
- **ЛА/Ао** - Соотношение легочной артерии к аорте
  ```
  ЛА/Ао = Легочная артерия / Аорта
  ```
- **В/А** - Соотношение вены к артерии
  ```
  В/А = Каудальная полая вена / Проксимальный венозный сегмент ЛА
  ```

### Нормализованные показатели
- **КДРн** - Нормализованный конечный диастолический размер
  ```
  КДРн = КДР / (Вес^0.294) / 10
  ```
- **КСРн** - Нормализованный конечный систолический размер
  ```
  КСРн = КСР / (Вес^0.315) / 10
  ```

### Функциональные показатели
- **FS** - Фракция укорочения (%)
  ```
  FS = (КДР - КСР) / КДР × 100%
  ```
- **FE** - Фракция выброса (%)
  ```
  FE = (КДО - КСО) / КДО × 100%
  ```

## 🧮 Математические формулы

### 📊 Основные расчеты

#### Соотношения размеров
```
ЛП/Ао = ЛПдиастола / Аорта
```
**Где:**
- ЛПдиастола - размер левого предсердия в диастолу (мм)
- Аорта - размер аорты (мм)

```
ЛА/Ао = Легочная артерия / Аорта
```
**Где:**
- Легочная артерия - размер легочной артерии (мм)
- Аорта - размер аорты (мм)

```
В/А = Каудальная полая вена / Проксимальный венозный сегмент ЛА
```
**Где:**
- Каудальная полая вена - размер КПВ (мм)
- Проксимальный венозный сегмент ЛА - размер ПВЛА (мм)

#### Нормализованные показатели
```
КДРн = КДР / (Вес^0.294) / 10
```
**Где:**
- КДР - конечный диастолический размер (мм)
- Вес - масса тела пациента (кг)
- 0.294 - коэффициент нормализации для КДР

```
КСРн = КСР / (Вес^0.315) / 10
```
**Где:**
- КСР - конечный систолический размер (мм)
- Вес - масса тела пациента (кг)
- 0.315 - коэффициент нормализации для КСР

#### Функциональные показатели
```
FS = (КДР - КСР) / КДР × 100%
```
**Где:**
- КДР - конечный диастолический размер (мм)
- КСР - конечный систолический размер (мм)
- FS - фракция укорочения (%)

```
FE = (КДО - КСО) / КДО × 100%
```
**Где:**
- КДО - конечный диастолический объем (мл)
- КСО - конечный систолический объем (мл)
- FE - фракция выброса (%)
